Vim Envionment
=============

Installation
-----------

	$ git clone git://github.com/winfred-lu/dotvim.git ~/.vim
	$ ln -sf ~/.vim/vimrc ~/.vimrc
	$ cd ~/.vim
	$ git submodule init
	$ git submodule update
	$ mkdir -p autoload; cd autoload
	$ ln -s ../bundle/pathogen/autoload/pathogen.vim

Usage
-----------

	$ vim
